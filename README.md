# Resume-Analyzer
AI Resume Analyzer is a web application that analyzes resumes against job descriptions using AI techniques. It extracts text from PDF resumes, compares skills and keywords with job requirements, and generates an ATS compatibility score with feedback to help candidates improve their resumes for better job opportunities.
